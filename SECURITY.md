# 安全策略

## 支持的版本

由于项目处于早期开发阶段，我们仅对最新版本提供安全更新支持。建议用户始终升级到最新发布版本（如有）。

| 版本状态                | 支持状态           | 说明                     |
| ----------------------- | ------------------ | ------------------------ |
| 主分支 (master)         | :white_check_mark: | 最新开发版               |
| 功能分支 (Xxx/xxx-xx)   | :white_check_mark: | 包含最新功能，不保证可用 |
| 归档版本 (exp/xxx-2xxx) | :x:                | 不提供安全更新支持       |

> **重要提示**：早期开发阶段版本可能存在未知安全问题，不建议在生产环境使用
